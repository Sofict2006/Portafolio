To do - Gestión de Tareas

Aplicación para la gestión de tareas, donde se puede crear, eliminar, y editar todo tipo de pendientes, los cuales se dividen de acuerdo a su estado (1 = completado, 0 = no completado).

Tecnologías Usadas
  • Frontend: HTML y CSS
  • Backend: PHP
  • Base de Datos: phpMyAdmin (MySQL)
Rol y Contribuciones

Sell & Work

Aplicación que permite a los usuarios publicar ofertas de empleo y a los trabajadores postularse. Guarda registro de los contratos realizados, y los documentos subidos, además posee un chat para la comunicación entre clientes y trabajadores.

Tecnologías Usadas
  • Frontend: HTML, CSS y Bootstrap
  • Backend: PHP
  • Base de Datos: phpMyAdmin (MySQL)
Rol y Contribuciones

Desarrollador backend y administradora de la base de datos.

EnergiTrack - Calculadora de Consumo Energético

Aplicación para la gestión del consumo energético en un espacio, donde en base al consumo de cada aparato de cada habitación del espacio, se calcula el gasto energético, el valor aproximado del gasto y se muestran recomendaciones.
